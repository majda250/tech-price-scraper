# Tech Price Scraper

Extraction automatique des prix de produits tech sur des sites e-commerce.

## Objectif
Surveiller les variations de prix et détecter les opportunités d'achat.

## Technologies
- Python 3
- Requests
- BeautifulSoup4
- CSV
